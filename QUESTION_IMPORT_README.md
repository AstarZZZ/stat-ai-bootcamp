# 题目添加与 JSON 批量导入说明

管理员登录后进入「题目管理」，可以通过两种方式添加题目：

1. 单题添加：填写题目标题、题目内容、参考答案，可选上传一张题目图片。
2. 批量导入：先新建或选择一个测验，再上传 `.json`、`.md`、`.markdown` 或 `.txt` 文件，一次导入多道题到该测验。

题目内容和参考答案按 Markdown 显示，支持表格、列表、代码块，以及 `$...$`、`$$...$$`、`\(...\)`、`\[...\]` 等 LaTeX 公式。

## JSON 文件格式

支持两种根结构。

第一种：直接使用数组。

```json
[
  {
    "title": "描述性统计练习",
    "content": "给定数据 1, 2, 3, 4, 5，请计算均值和样本方差。",
    "answer": "均值为 3，样本方差为 2.5。"
  },
  {
    "title": "Python 列表练习",
    "content": "写一个函数，返回列表中的最大值和最小值。",
    "answer": "可使用 max(nums) 和 min(nums)，也可以手写循环。"
  }
]
```

第二种：使用 `questions` 字段。

```json
{
  "questions": [
    {
      "title": "线性回归概念",
      "content": "请解释最小二乘法的目标函数。",
      "answer": "最小化预测值与真实值残差平方和。"
    }
  ]
}
```

## 字段说明

必填字段：

- `title`：题目标题
- `content`：题目内容

选填字段：

- `answer`：参考答案，仅管理员可见
- `imagePath`：图片地址，例如 `/uploads/demo.png` 或可访问的外链

## 兼容字段名

为了方便从其他系统导入，也兼容以下字段名。

- 标题：`title`、`name`、`questionTitle`、`题目标题`、`标题`
- 内容：`content`、`question`、`stem`、`body`、`题目内容`、`题干`、`问题`
- 答案：`answer`、`referenceAnswer`、`solution`、`参考答案`、`答案`、`解析`
- 图片：`imagePath`、`imageUrl`、`image`、`图片`、`图片地址`

## 限制

- 文件可以是 `.json`、`.md`、`.markdown` 或 `.txt`。系统会先尝试解析 JSON，失败后再尝试按 Markdown/TXT 解析；如果都无法识别，会提示解析失败。
- 单个导入文件最大 2 MB。
- 单次最多导入 200 道题。
- JSON 批量导入不上传图片文件本体；如果题目需要新图片，请用单题添加上传图片，或在 JSON 里填写已有图片地址。

## Markdown / TXT 文件格式

使用一级标题 `#` 或 `第 n 题：...` 分隔题目；使用 `## 参考答案`、`## 答案` 或 `## 解析` 分隔参考答案。

```md
# 第 1 题：Sigmoid 公式

已知 $f(x)=\frac{1}{1+e^{-x}}$，请写出导数。

## 参考答案

$f'(x)=f(x)(1-f(x))$
```

如果需要引用已有图片地址，可以在题目段落中单独写一行：

```text
imagePath: /uploads/demo.png
```

## 操作步骤

1. 使用管理员账号登录。
2. 进入「题目管理」。
3. 新建测验，或点击已有测验。
4. 在「批量导入题目」区域选择 JSON、Markdown 或 TXT 文件。
5. 点击「批量导入到当前测验」。
6. 页面提示导入数量后，当前测验的题目列表会自动刷新。
